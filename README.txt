== Description ==

Este plugin permite que su tienda pueda utilizar el sistema de tracking, tarificación y envíos que ofrece nuestra API.

Contenido:
    - manual general plugin (instalacion, desinstalacion, uso)
    - comprimido con archivos para instalacion del plugin.
